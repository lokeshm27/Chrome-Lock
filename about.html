About Chrome Lock

Chrome Lock is an extension made to fulfill your needs secure your browser. There are many things that differentiates Chrome Lock from other extensions of same category such as

  • Security

    We worked hard on this extension to make this extension as secure as possible. This extension is more secure than other extensions, To give one example, Unlike other extensions the password is not stored raw. Since, Chrome will not encrypt the extension data and it's not difficult to access an extension's data. We implemented best methods to make this extenion secure, The advanced protection, but that may use more system resources, so We added an option to enable the Advanced protection. With Advanced protection by passing the extensions security won't be a piece of cake.

  • Design

    We have designed both looks (front-end) and functionality (back-end) of the extension as good as possible. We made sure that the design of the extension won't trouble you. We have emphasized on small details without which it will hard to work around the extension. This extension consits of many web-pages. We have designed the web pages as good as possible without taking much time.

  • Options and Features

    Chrome Lock extension come with many options. We have given you the choice to select how extension should behave in some conditions without effecting the functionality of the extension. We have used the best method availbale to implement a feature, But Some features are still in beta-stage. Beta-Stage features works but it means that we have not yet tested its working in all possible conditions yet. We will remove the beta-stage warning as soon as the feature passes the tests.

  • Permissions

    Chrome Lock requires only limited permission and we promise the we won't store your information in any manner. The permissions required and their need is listed below :

    • Tabs and Active Tab permission required to manipulate Tabs and Windows.
    • Storage permission required to store password, password hint and other data.
    • Alarms permission required for advanced protection. This permission enables the extension to run some code at specified time intervals.
    • Idle permission required to check chrome's idle state. Used in Inactivity Timeout Feature.
    • Context Menu permission is required to add lock option in chrome right-click Menu ( Context Menu ).
    • Notifications permission is required to show notifications. Chrome Lock shows a notification when Auto Lock is disabled. If you don't want the notification to show, you can block notifications from the extension.
