If you are watching a video or listening to a audio, and you don't want to lock chrome automatically due to inactivity, Then use this option.
You can select 1st option which disable automatic lock when a audio is playing in current tab irrespective of the site. If you want to disable auto lock only when audio is playing in selected sites, choose 2d option. This option checks the URL of current tab which is playing audio. If the domain of the tab matches with the domain listed above, Auto lock will be disabled, And a Notification is shown.

Note : This feature is still in beta-stage, May not work perfectly.
There is no optimal way to detect if video is playing in a tab. So, Using Audio this feature is implemented.
If audio is muted by any means, It is considered as a tab without Audio and this option will not work.
